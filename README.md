# supplyManage

Welcome to supplyManage! This is a simple simulation for a sample, mini, non-yet-scalable supply chain. Right now this is a pre-alpha release. That is why chain-saving is not yet ready.

## To Install and Setup
1. Clone the Rep.
2. cd to folder
3. ```python3 main.py```
